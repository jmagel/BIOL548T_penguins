DIRECTORY INFORMATION

This folder houses the following files:

01_penguins_data_save.R – Script to extract and save necessary data from the 'palmerpenguins' R package
(saved to 'penguins.csv' in the 00_rawdata directory)

02_penguins_figures.R – Script to create and save Figures 1 and 2 (see 'figure_01.jpg' and 'figure_02.jpg' 
in the 02_figures directory)
