DIRECTORY INFORMATION

This folder houses the following files:

BIOL548T_penguins.bib – BibTeX file containing references from the 'BIOL548T_penguins' Zotero library

canadian-journal-of-zoology.csl – Citation Style Language (CSL) file specifying the formatting of references according to the 
style used by the Canadian Journal of Zoology

grateful-refs.bib – BibTeX file containing citation information for the R packages referenced in the 'penguins_manuscript.Rmd' file

mageletal_penguins_ms.pdf – PDF of the manuscript, rendered using the 'penguins_manuscript.Rmd' file

penguins_manuscript.Rmd – R markdown file containing code necessary to render a PDF version of the manuscript
