DIRECTORY INFORMATION

This folder houses the following files:

_DATA_DICTIONARY.txt – Data dictionary containing a more detailed description of the penguins.csv data file, as well as an explanation of the 
individual variables found within the data frame.

penguins.csv – .csv file containing the data that was used to create all figures and conduct all statistical analyses associated with this 
project. This file was created using the '01_penguins_data_save.R' script in the 01_scripts folder to download and save the 'penguins' data 
frame from the 'palmerpenguins' R package.
